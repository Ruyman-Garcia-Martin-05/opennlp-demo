\doxysubsection{Detailed Description}
Clase principal para demostrar el etiquetado de partes del discurso (POS) con Open\+NLP. 

Esta clase utiliza un modelo preentrenado para etiquetar las partes del discurso de un conjunto de tokens y mostrar las probabilidades asociadas a cada etiqueta.
