\doxysubsection{Detailed Description}
Clase para entrenar un modelo de reconocimiento de nombres con Open\+NLP. 

Utiliza datos de entrenamiento para generar un modelo que puede reconocer nombres propios (entidades nombradas) en texto.

\begin{DoxyNote}{Note}
Los datos de entrenamiento deben contener al menos 15,000 oraciones para generar un modelo con buen desempeño. 
\end{DoxyNote}
