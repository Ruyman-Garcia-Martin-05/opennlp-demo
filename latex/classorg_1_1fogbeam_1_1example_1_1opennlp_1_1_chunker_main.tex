\doxysubsection{Detailed Description}
Clase principal para demostrar el uso del modelo de chunking de Open\+NLP. 

Esta clase utiliza el modelo de chunking para analizar una oración y asignar etiquetas de chunks (trozos) a las palabras basándose en las partes del discurso (POS tags).
